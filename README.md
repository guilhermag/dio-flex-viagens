## Como testar o projeto

### Opção 1: Clone o repositório e abra remotamente

Digite no seu terminal

``
git clone https://github.com/guilhermag/dio-flex-viagens.git
``

``
cd dio-flex-viagens/
``

Agora é só abrir o ``index.html``.
